# Roadmap

## Near-Term

- Expand proof-based validation for optional services.
- Add reproducible smoke-test automation for required local path.
- Improve service-level observability documentation.
- Standardize local bootstrap behavior across all Node services.

## Mid-Term

- Expand architecture diagrams with service contracts and event schema lineage.
- Add platform-level integration test scenarios for core detection-to-alert flow.
- Formalize release notes and changelog process for monorepo milestones.

## Long-Term

- Harden production deployment guidance and compliance evidence workflows.
- Add benchmark-driven capacity planning docs for event throughput.
- Broaden multi-environment deployment templates with reproducible promotion paths.

## Status Transparency

Roadmap items are planning signals, not guaranteed delivery commitments.
