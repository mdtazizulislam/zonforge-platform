Production schema hotfix date: 2026-04-08

Scope:
- Fix Railway production backend crash caused by schema drift during startup.
- Crash source from Railway logs: Postgres 42703 on missing auth_refresh_tokens.session_id while creating ix_auth_refresh_tokens_session.

Local versus production schema comparison:
- Local backend initDatabase expected auth_refresh_tokens.session_id and created ix_auth_refresh_tokens_session.
- Production already contained detections, detection_events, and risk_scores columns used by computeRiskStats.
- Production was missing auth_refresh_tokens.session_id.
