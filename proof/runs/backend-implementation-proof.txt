API ENDPOINTS READY
───────────────────

Authentication:
  POST /auth/register
    Request: { email: "user@example.com", password: "secure" }
    Response: { success: true, userId: 1, token: "jwt...", redirectUrl: "/success" }
    Status: 400 on error

  POST /auth/login  
    Request: { email: "user@example.com", password: "secure" }
    Response: { success: true, userId: 1, token: "jwt..." }
    Status: 400 on error

Protected API:
  GET /api/user
    Headers: Authorization: Bearer <JWT_TOKEN>
    Response: { user: { id: 1, email: "user@example.com" } }
    Status: 401 if unauthorized

Webhooks:
  POST /webhook/stripe
    Headers: stripe-signature: <STRIPE_SIGNATURE>
    Body: Raw Stripe event data
    Handles: checkout.session.completed
    Creates: User, Tenant, Subscription on payment

Utility:
  GET /health
    Response: { status: "ok" }

Build for Production:

Test Endpoints:
  # Health check
  curl http://localhost:3000/health

  # Register user
  curl -X POST http://localhost:3000/auth/register \
    -H "Content-Type: application/json" \
    -d '{"email":"test@example.com","password":"test123"}'

  # Login user  
  curl -X POST http://localhost:3000/auth/login \
    -H "Content-Type: application/json" \
    -d '{"email":"test@example.com","password":"test123"}'

  # Access protected endpoint (replace TOKEN)
  curl http://localhost:3000/api/user \
    -H "Authorization: Bearer TOKEN"
